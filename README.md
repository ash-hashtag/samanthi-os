
Requirements:

- qemu-system-x86-64
- rust nightly + cargo


How To Run:
` cargo run `



Features:
- Memory Filesystem not tempfs
- Heap Allocation support
- Can kinda edit text
- Can kinda see images
- No syscalls (yet)
- No user level programs support (yet)
- No multithreading (yet)

- https://os.phil-opp.com/
